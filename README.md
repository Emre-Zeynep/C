# Compiler
**C++ Komut:** *Cmd üzerinde derlemek için komut* <br />
  >g++ -o 01 01.cpp
  
**C Komut:** *Cmd üzerinde derlemek için komut* <br />
  >gcc -o 01 01.c
  
**C++ Komut:** *Cmd Üzerinde Birden fazla cpp dosyası derlemek için komut* <br />
  >g++ -c Main.cpp Test.cpp
  
**C++ Komut:** *Birleştirmik için komut* <br />
  >g++ Main.o Test.o -o main
